重复字符串(如abc=>abcabc)

```js function repeat(target,n){ return (new Array(n+1).join(target)); } ```

打乱数字数组的顺序

```js var numbers = [5, 458 , 120 , -215 , 228 , 400 , 122205, -85411]; numbers = numbers.sort(function(){ return Math.random() - 0.5}); /* numbers 数组将类似于 [120, 5, 228, -215, 400, 458, -85411, 122205] */ ```

在条件中使用逻辑与或

```js var foo = 10; foo == 10 && doSomething(); // is the same thing as if (foo == 10) doSomething(); foo == 5 || doSomething(); // is the same thing as if (foo != 5) doSomething(); ```
